# Bloodlore Glossary

A web-based item glossary viewer for browsing and managing game items with an interactive interface.

## Features

- Search and filter items by effects, name, region, and source
- Equip/unequip items and manage builds
- Save and load builds
- Light/Dark theme toggle
- Categorized item browsing

## Usage

Simply open `index.html` in a web browser to use the glossary viewer.

## Documentation

- [Using GitHub Copilot on Android](./COPILOT_ANDROID.md) - Guide for using GitHub Copilot on Android tablets

## Development

This is a static HTML/CSS/JavaScript application with no build step required. Edit the files directly and refresh your browser to see changes.
